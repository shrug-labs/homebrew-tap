# shrug-labs Tap

## How do I install these formulae?

```bash
brew install shrug-labs/tap/aipack
```

Or tap the repository first and then install:

```bash
brew tap shrug-labs/tap
brew install aipack
```

## Included formulae

- `aipack` — portable AI agent configuration pack manager from `shrug-labs/aipack`

## Notes

- Release assets are sourced from GitHub Releases in `shrug-labs/aipack`.
- The formula currently installs raw release binaries for macOS (`arm64`, `amd64`) and Linux (`amd64`).
- This tap was previously published as `dfoster-oracle/homebrew-tap`; the old name redirects here automatically, but re-tapping with the new name is recommended:

  ```bash
  brew untap dfoster-oracle/tap || true
  brew tap shrug-labs/tap
  ```
