# dfoster-oracle Tap

## How do I install these formulae?

```bash
brew install dfoster-oracle/tap/aipack
```

Or tap the repository first and then install:

```bash
brew tap dfoster-oracle/tap
brew install aipack
```

## Included formulae

- `aipack` — portable AI agent configuration pack manager from `shrug-labs/aipack`

## Notes

- Release assets are sourced from GitHub Releases in `shrug-labs/aipack`.
- The formula currently installs raw release binaries for macOS (`arm64`, `amd64`) and Linux (`amd64`).
